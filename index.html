PM Kisan eKYC Mobile Se Kaise Kare 2025 — Complete Step-by-Step Guide

PM Kisan eKYC mobile से करना आज बहुत आसान है। इस guide में हम mobile से Aadhaar OTP के साथ eKYC और नए Face Authentication तरीके को step-by-step समझाएँगे — साथ में troubleshooting और FAQs भी दिए गए हैं।

PM Kisan eKYC — क्या है और क्यों ज़रूरी?

eKYC यानी electronic Know Your Customer — इसका मकसद सरकार को digitally verify करना है कि लाभार्थी सही है। अगर आपका eKYC पूरा नहीं हुआ है तो PM Kisan की किस्तें रोक दी जाती हैं।

Quick note: अगर Aadhaar में mobile number link नहीं है तो आप Face Authentication से भी eKYC कर सकते हैं — नीचे पूरा तरीका दिया गया है।

Mobile से PM Kisan eKYC कैसे करें — Step-by-step

Follow these exact steps:

  1. Official website खोलें: https://pmkisan.gov.in — Mobile browser में Desktop mode ON करें (यदि कुछ विकल्प नहीं दिख रहे हों)।
  2. Home page पर "e-KYC" विकल्प चुनें: आधिकारिक पेज पर आपको e-KYC (Aadhaar OTP) और Face Authentication दोनों विकल्प मिलते हैं।
  3. Aadhaar Number डालें: अपना 12-digit Aadhaar डालें और Get OTP पर क्लिक करें।
  4. OTP Enter करें: Aadhaar से linked mobile number पर आए OTP को डालकर verify कर दें। Successful होने पर eKYC instantly complete हो जाएगा।

NEW (2025) — Face Authentication (Aadhaar mobile number न होने पर)

अगर Aadhaar में मोबाइल नंबर linked नहीं है तो face-auth का तरीका use कर सकते हैं:

  • Play Store से PM Kisan Face Authentication app डाउनलोड करें।
  • App खोलकर Aadhaar number डालें और front-camera से अपना face scan करें।
  • Matching successful होने पर eKYC update हो जाएगा।

eKYC Status कैसे चेक करें?

PM-Kisan site → Beneficiary Status पर जाएँ, यहाँ आप Aadhaar / Account / Mobile number डालकर status देख सकते हैं।

Quick Tips to Get eKYC Done Fast

  • Chrome incognito mode use करें — इससे cache/old session interfere नहीं करेगा।
  • Server busy होने पर off-peak hours में (रात) try करें।
  • Aadhaar और bank में same name/status रखें।
  • Face auth के लिए अच्छा light और clear camera angle रखें।

FAQ — अक्सर पूछे जाने वाले सवाल

क्या eKYC न होने पर किस्त रुक जाएगी?

हाँ — यदि eKYC pending है तो किस्त pending/on-hold में दिख सकती है। सबसे पहले eKYC और bank seeding check करें।

Face Authentication क्या सुरक्षित है?

Face Authentication UIDAI के guidelines के अनुरूप काम करता है; हमेशा आधिकारिक app या website का ही उपयोग करें।

यदि OTP नहीं आता तो क्या करें?

Retry करें, network बदलकर try करें, या face authentication का option use करें।

Face Auth के लिए कौन सा ऐप भरोसेमंद है?

सिर्फ सरकारी या UIDAI endorsed apps ही उपयोग करें; unofficial apps से बचें।

कितने समय में eKYC reflect होगा?

अमूमन OTP-verify होने पर instant; face-auth में कुछ घंटे लग सकते हैं।

Bank seeding कैसे चेक करें?

PM Kisan official portal पर Beneficiary section से account detail डालकर check करें।

अगर नाम mismatch हो तो क्या करें?

सबसे पहले Aadhaar में correction करें या बैंक में नाम update कराएँ।

क्या किसी third-party से eKYC करवाना सुरक्षित है?

पासवर्ड/OTP किसी के साथ share मत करो — केवल official portal/app का इस्तेमाल करो।

Face Auth बार-बार fail हो रहा है — solution?

Better lighting, camera cleaned, stable internet और official app use करें।

क्या mobile number change होने से समस्या आती है?

हाँ — Aadhaar में registered mobile number mismatch होने पर OTP नहीं आएगा; तब face-auth use करो या Aadhaar update कराओ।

Conclusion

PM Kisan eKYC 2025 में mobile से काफी आसान और fast हो गया है — Aadhaar OTP से instant verification possible है और अगर mobile number linked नहीं है तो Face Authentication विकल्प है। अगर आपकी किस्त रुकी हुई है तो सबसे पहले eKYC और bank seeding verify करें।
